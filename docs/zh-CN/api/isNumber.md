isNumber
========

检查数据是否为有效数字

案例
----

### 检查-123.4是否为数字
```php
if (widget()->isNumber(-123.4)) {
    echo 'Yes';
} else {
    echo 'No';
}
```

#### 运行结果
```php
'Yes'
```

调用方式
--------

### 选项

| 名称              | 类型    | 默认值                           | 说明                                             |
|-------------------|---------|----------------------------------|--------------------------------------------------|
| notNumberMessage  | string  | %name%必须是有效的数字           | -                                                |
| negativeMessage   | string  | %name%不能是数字                 | -                                                |

### 方法

#### isNumber($input)
检查数据是否为有效数字
